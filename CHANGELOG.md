# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Changed
- Updated minimum Go version to 1.22

## [0.1.0] - 2024-12-29

### Added
- Initial release
- Configuration-driven data mapping
- Zero dependencies on domain objects
- Complete abstraction from data sources
- Pluggable adapter architecture
- Built-in filesystem adapter for JSON data
- Environment variable substitution in configuration
- Custom actions and hooks
- Transaction support
- Bulk operations
- CQRS patterns (read/write separation, event sourcing)
- Comprehensive test coverage (80.6%)
